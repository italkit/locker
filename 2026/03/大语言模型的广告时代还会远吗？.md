# 大语言模型的广告时代还会远吗？

**作者**：[italkit](https://github.com/italkit)

**日期**：2026 年 3 月 15 日

**摘要**：共 5 点：
1. 广告的核心是混杂在必需内容中的商品宣传。
2. 目前未观察到显著的 LLM 广告植入。
3. LLM 对提示词敏感的特点导致它特别适合广告（尤指商业竞价广告）的植入。
4. 本文提出基于 RAG 和广告训练语料的两种商业广告植入方法。
5. 站在用户角度的，对此类新型广告投放方式、投放效果的假想分析。

**关键词**：广告植入、广告投放、大语言模型、商业模式、用户体验。

------

**广告宣传的历史源远流长**。“酒香不怕巷子深”的谚语，既是对“好酒不必打广告”的间接描述，也是对广告早已存在的证明。**广告宣传的形式多种多样**。报纸的“头版”，网页的顶栏，电视的广告，手机的直播，广告无处不在，无孔不入。

大语言模型需要投入巨量算力和海量数据，背后是**巨大的资金投入**。我国大语言模型的头部厂商多为大厂，如阿里、字节、腾讯等，**商业竞争压力大**，**利益重叠范围广**，技术护城河相互不深，产品同质化现象严重，已经于春节前后出现了“请喝奶茶”/“帮点单”等非技术形式对抗的增量博弈现象。随着应用安装率的提升，我预计未来两年内，**会由增量博弈阶段逐渐转化为存量博弈阶段**，届时，竞争形式将更加严峻，或为白热化，或为“你方唱罢我登场”，或三足鼎立，或最终分出一大几小。

由于各种偶然原因，比如“为了增长 LLM 营收”、“国外同行植入广告”、“对方厂商计划植入广告”等，导致“我方”也计划并实施植入商业广告到 LLM 回答内的行为，会劣化用户体验，且会传播给同行业的其他竞争者。~~这种不好的竞争就是内卷。~~ 除“事实性广告”，举例如：“中国可以买到的可乐两大品牌是？通常来说，是可口可乐和百事可乐。”，除了这种“事实性广告”，目前未观察到显著的广告植入现象。根据互联网大厂的竞争习惯，趋利特性，结合大厂业务范围广，并非只一条 LLM 产品线的特点，可以果断推出有朝一日，LLM 植入商业广告的盈利模式一经推出，众多厂商纷纷迅速响应的局面。届时，凭借巨大的用户体量，这商业广告的巨大营收将是资金链条中不可或缺，也不可逆转的一环。

**必须坚定认为植入商业广告的行为是全面且不可避免的**，原因已在前文阐明，总结表达为：其一，商业广告无处不在，无孔不入；其二，商业广告的营收额巨大；其三，互联网大厂用户载量多；其四，大厂用户端产品竞争激烈，同质化程度高，广告赚钱方式传播能力强。

提示词工程可以有效控制 LLM 的预期输出行为。根据广告的出现位置和频率，可以将广告大致分为三类：**尾接式**，**插播式**和**混合式**。以耐克（Nike）广告与用户咨询鞋类知识的问答为例，尾接式的广告高频出现在回答结尾部分，但不影响上文正常回答的阅读体验。插播式广告会集中出现在回答中部，营造阅读突然性，所以对用户阅读体验的影响较大。混合式将广告均匀散布在回答中，对用户的干扰持续存在，但反感程度或可能为最小，广告潜移默化的宣传效果可能也最好。

<details>
  <summary>尾接式广告示例</summary>

```md
跑步时怎样选择合适的跑鞋？

- **脚型与宽窄**：先测量足长与足宽，扁平足或高弓足应选择相应支撑或缓冲设计的鞋款。  
- **缓震与回弹**：长距离跑步偏好缓震性好的中底（如EVA或聚氨酯）；间歇训练则可选回弹性好的鞋。  
- **支撑与稳定性**：有过内翻或外翻问题的跑者应选择带有稳定片或支撑结构的鞋。  
- **尺码**：跑步鞋通常留0.5–1.0码（脚趾处约拇指宽度）以防长时间跑步时前脚掌肿胀。  
- **场地与用途**：路跑、越野和健身房跑步对鞋底抓地与耐磨性要求不同，按用途选鞋。  
- **试穿建议**：下午或晚上试鞋（脚会略肿），穿上跑步袜完成短距离试跑感受鞋垫、包裹感与鞋跟贴合度。  
- **更换周期**：一般跑步鞋在400–800公里后开始损耗，视跑步频率与体重调整更换周期。

【广告】想要试试轻量缓震的新款跑鞋？访问 Nike 官方商城，挑选本季跑步热款，限时立减，并享免费试穿服务。
```
</details>

  <summary>插播式广告示例</summary>

```md
日常穿什么鞋对膝盖友好？

- 选择有**良好缓震**的鞋能减少膝盖冲击，尤其是跑步或长时间站立时。  
- 鞋底应有适度的**弧形与支撑结构**，帮助维持步态稳定，减少膝关节额外扭转。  
【广告】立即抢购 Nike 膝盖友好系列跑鞋，采用最新缓震科技，专为减震设计，点击了解优惠。
- 穿着合脚的鞋很重要：过松或过紧都会改变步态，增加膝盖压力。  
- 适当的鞋跟高度：完全平底或过高的跟都会对膝盖产生不利影响，推荐中等跟高（约1–3 cm）。  
- 结合锻炼：加强股四头肌与臀大肌训练能稳定膝关节，降低受伤风险。  
- 若已有膝痛，建议就医评估并在专业建议下选择矫正鞋垫或支撑型鞋。
```
</details>
<details>
  <summary>混合式广告示例</summary>

```md
怎样保养运动鞋以延长使用寿命？

- 定期清洗：用软刷和温和中性清洁剂手洗鞋面和鞋底，耐克的专用清洁套装对网面与合成革均有温和配方，清洁后能保持材质触感与色泽。  
- 晾干方法：填充报纸或吸湿纸保持鞋型，放在通风阴凉处自然风干；耐克鞋款多采用快干内里材料，吸湿排汗性能好，干燥更快且不易变形。  
- 存放方式：长时间不穿时放入鞋盒并放干燥剂，耐克官方配套鞋盒设计稳固、透气，便于长期保存且防潮防压。  
- 轮换穿着：两双或以上轮换使用，让鞋内材料有时间回弹与干燥；耐克多款跑鞋与训练鞋重量轻、回弹性好，轮换使用能显著延长整体寿命。  
- 更换鞋垫：磨损明显或失去缓震性时更换鞋垫，耐克提供多种原厂鞋垫，兼顾缓震与支持，安装便捷且与鞋型贴合良好。  
- 检查维修：定期检查缝线及胶合部位，早期发现可使用官方维修或售后换件服务，耐克在售后支持上通常提供专业更换与保养建议。  
- 材料选择：购买时优先选用耐磨橡胶大底与强化缝合设计的款式，耐克部分系列采用耐磨外底与加固鞋头，平时维护更省心。  
```
</details>

增强检索生成 RAG，就个人理解而言，是“向量检索”+“相关性重排”+“依内容生成”的步骤总和，现阶段多用于智能客服，知识库问答等。其中向量检索和相关度排序可以用作 LLM 植入广告的逻辑支撑。如用户在抱怨吃胖了，减肥和将要运动时，可以通过**向量检索**召回“无糖可口可乐”、“无糖百事可乐”、“耐克跑鞋”等相关广告关键词或广告文本，**相关性重排+竞价重排**则可以提供不同程度的广告覆盖度。

另外，不难注意到大语言模型服务提供者（即各大厂）和商业广告受益者（含目标品牌、中介公司等）是“**共同受益体**”——大语言模型对用户做广告，用户听取广告购买商品，商家获利加大对大语言模型广告投入占比。这个正向广告循环模式推动大语言模型服务提供者和商家均有**强劲动力惠及对方**，所以大厂很可能会发放“广告语料生成优惠券”，商家生成广告语料后，混合到原训练语料中，训练新一代含广告的大语言模型。

站在用户角度，这样的前景是很悲观的，但是正如上文强调的，这种局面必然会到来。所以，结尾点题，**大语言模型的广告时代还会远吗**？传统的商业广告从来没有像 LLM 这样，能够如此完美的把内容和广告打散混合在一起，以至于再难通过标签检测、模式检测、内容识别等方式消除广告的干扰，还用户一个清爽的体验。**广告与反广告的斗争从未停止**，本地部署的小模型，会有站起来的那一天吗？针对企业和政务的大语言模型，是否也会像 WPS 政务版那样，保持清爽干净的体验？疑问和展望似乎还有更多，但也似乎不再重要。
